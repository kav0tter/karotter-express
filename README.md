# karotter-express

[![License: MIT](https://img.shields.io/badge/License-MIT-red.svg)](LICENSE)
[![Chrome](https://img.shields.io/badge/Chrome-Supported-blue.svg)](https://www.google.com/chrome/)
[![Firefox](https://img.shields.io/badge/Firefox-Supported-orange.svg)](https://www.mozilla.org/firefox/)

**karotter.com** をキーボードショートカットで高速操作するためのブラウザ拡張機能です。Vim 風や Twitter/X 風のプリセットを標準搭載し、すべてのキーバインドをカスタマイズ可能です。

---

## 🎯 特徴

- ⚡ **高速ナビゲーション** - タイムラインの投稿を矢印キーや `j`/`k` で素早く移動
- 🎨 **5 つのプリセット** - デフォルト、Vim 風、Twitter/X 風、Claude 作、v0 作から選択可能
- 🔧 **フルカスタマイズ** - すべてのアクションに任意のキーを割り当て可能
- 🎭 **クイックリアクション** - テンキーなどで即座に絵文字リアクション
- 👁️ **インラインヒント** - 投稿のボタン上にショートカットキーを表示
- 📱 **フローティングパネル** - 画面右下にキー一覧を常時表示
- ⌨️ **2 ストロークキー** - `gh` のような連続入力にも対応
- 🌐 **マルチドメイン** - karotter.com / .jp / .karon.jp すべてに対応

## 📦 インストール

### Chrome / Edge 系

1. このリポジトリをクローンまたはダウンロード
   ```bash
   git clone https://github.com/szk27/karotter-express.git
   ```

2. Chrome の拡張機能ページ (`chrome://extensions/`) を開く

3. 右上の「デベロッパーモード」を有効化

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. `chrome/` ディレクトリを選択

### Firefox

1. このリポジトリをクローンまたはダウンロード
   ```bash
   git clone https://github.com/szk27/karotter-express.git
   ```

2. Firefox のアドオンページ (`about:debugging#/runtime/this-firefox`) を開く

3. 「一時拡張機能を読み込む」をクリック

4. `manifest.firefox.json` を選択

## ⌨️ デフォルトキーバインド

### ナビゲーション

| キー | アクション |
|------|-----------|
| `↓` / `ArrowDown` | 次の投稿へ移動 |
| `↑` / `ArrowUp` | 前の投稿へ移動 |
| `Enter` | 投稿を開く |

### 投稿アクション

| キー | アクション |
|------|-----------|
| `l` | いいね |
| `r` | リポスト（直接） |
| `q` | 引用リポスト |
| `b` | ブックマーク |
| `@` | リプライ |
| `p` | 投稿者プロフィールを開く |
| `o` | 引用元投稿を開く |

### グローバル操作

| キー | アクション |
|------|-----------|
| `n` | 新規投稿 |
| `/` | 検索欄にフォーカス |
| `?` | ヘルプの表示/非表示 |
| `[` | 前の内側タブへ |
| `]` | 次の内側タブへ |
| `{` | 前の外側タブへ |
| `}` | 次の外側タブへ |

### ページ移動

| キー | アクション |
|------|-----------|
| `1` | ホームへ移動 |
| `2` | 検索ページへ移動 |
| `3` | 通知へ移動 |
| `4` | メッセージへ移動 |
| `5` | ブックマークへ移動 |
| `6` | 自分のプロフィールへ移動 |
| `7` | 設定へ移動 |

## 🎨 プリセット

### 利用可能なプリセット

| プリセット名 | 特徴 |
|-------------|------|
| **デフォルト** | 矢印キーと数字キーを使った標準設定 |
| **Vim 風** | `j`/`k` で移動、`g` + キーでページ移動 |
| **Twitter / X 風** | Twitter のショートカットに似た設定 |
| **Claude 作** | ホームポジションを意識した配置 |
| **v0 作** | 独自の効率重視レイアウト |

### カスタムプリセットの作成

1. 拡張機能の設定画面を開く
2. キーバインドをカスタマイズ
3. 「現在の設定を保存」ボタンをクリック
4. プリセット名を入力して保存

保存したプリセットはいつでも切り替え可能です。

## 🛠️ 設定方法

拡張機能のアイコンをクリックするか、ブラウザの拡張機能管理から設定画面を開けます。

### 3 つのタブ

#### 1. Keybindings (キーバインド)

- プリセットの選択と切り替え
- 各アクションのキー変更
- カスタムプリセットの保存・削除
- デフォルトへのリセット

#### 2. Reactions (クイックリアクション)

- テンキー (`Numpad1`〜`Numpad9`) に絵文字を割り当て
- 投稿フォーカス中にワンキーでリアクション

#### 3. Display (表示オプション)

- **インラインヒント**: 各投稿のボタン上にショートカットキーを表示
- **フローティングパネル**: 画面右下にキー一覧を常時表示

## 💡 使い方

### 基本操作

1. **投稿のフォーカス**: `↓`/`↑` または `j`/`k` で投稿間を移動
2. **アクション実行**: フォーカス中の投稿に対して `l` (いいね)、`r` (リポスト) など
3. **ヘルプ表示**: `?` でショートカット一覧を表示

### Esc キーの便利な使い分け

Esc キーは文脈に応じて異なる動作をします：

1. 入力欄フォーカス中 → 入力を確定してフォーカスを外す
2. ヘルプ表示中 → ヘルプを閉じる
3. モーダル表示中 → モーダルを閉じる
4. 投稿フォーカス中 → フォーカスを解除
5. それ以外 → 前のページへ戻る

### 2 ストロークキー

Vim 風プリセットなどでは、`gh` (ホーム)、`gs` (検索) のような 2 キーの連続入力に対応しています。1 打鍵後 500ms 以内に次のキーを入力すると認識されます。

## 🏗️ 技術情報

### アーキテクチャ

- **Manifest V3** (Chrome) / **Manifest V2** (Firefox) 対応
- **Service Worker** バックグラウンド処理
- **Content Scripts** DOM 操作とキーイベントフック
- **Shadow DOM** ヘルプオーバーレイとフローティングパネル

### ファイル構成

```
karotter-express/
├── chrome/                    # Chrome 用拡張機能
│   ├── manifest.json          # マニフェスト (V3)
│   ├── background.js          # サービスワーカー
│   ├── content-script.js      # メイン処理
│   ├── defaults.js            # デフォルト設定・プリセット
│   ├── selectors.js           # DOM セレクタ定義
│   ├── help-overlay.js        # ヘルプ表示
│   ├── inline-hints.js        # インラインヒント表示
│   ├── floating-panel.js      # フローティングパネル
│   ├── page-reaction.js       # 絵文字リアクション処理
│   ├── options.html           # 設定画面 HTML
│   ├── options.css            # 設定画面スタイル
│   ├── options.js             # 設定画面ロジック
│   └── help-overlay.css       # ヘルプ/ヒントスタイル
├── manifest.firefox.json      # Firefox 用マニフェスト (V2)
├── icons/                     # アイコンファイル
└── README.md
```

- 各 www サブドメイン
